# Jeu-du-Snake---Python-POO---L1-Informatique
Projet réalisé en **L1 Informatique**, ce dépôt contient une implémentation du célèbre jeu du Snake en utilisant Python et la bibliothèque Tkinter pour l'interface graphique.

Le jeu du Snake consiste à contrôler un serpent qui grandit en mangeant de la nourriture apparaissant aléatoirement sur la grille.  
Le but est de faire le meilleur score possible sans heurter les murs ou son propre corps.

---

## 🧰 Technologies utilisées

- Python 3
- Programmation orientée objet 
- Tkinter (interface graphique)
